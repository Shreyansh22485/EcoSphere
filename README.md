## 🌟 Project Overview

EcoSphere transforms traditional e-commerce by integrating sustainability metrics, AI-powered product analysis, and community-driven environmental impact tracking. Users earn **Impact Points** for sustainable purchases, compete in eco-challenges, and participate in group buying for better environmental outcomes.

### 🎯 Key Innovation
- **AI-Powered EcoScore System**: Google Gemini AI analyzes products for comprehensive sustainability scoring (0-1000 scale)
- **Real-time Impact Tracking**: Track CO2 saved, water conserved, and waste prevented
- **Gamified Sustainability**: Tier-based progression system with rewards and badges
- **Community Group Buying**: Collective purchasing for better prices and environmental impact

## 🏗️ System Architecture

- **Products**: EcoScore, sustainability metrics, inventory
- **Groups**: Community management, group buying

- **Partners**: Business profiles, product management

### AI Integration
- **Google Gemini AI**: Product analysis and scoring

- **Impact Forecasting**: Predictive environmental analytics
- **Quiz Generation**: Interactive learning experiences

### API Structure
```
/api
├── /auth          # Authentication & authorization
├── /products      # Product management & EcoScore
├── /users         # User profiles & impact tracking

├── /orders        # Purchase processing & fulfillment
```

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB (v5+)
- Google Gemini AI API Key
- Cloudinary Account

### Installation

1. **Clone the repository**
```bash
git clone [repository-url]
cd Amazon\ HackOn
```

2. **Install dependencies**
```bash
# Install frontend dependencies
npm install

# Install backend dependencies
cd ecosphere-backend
npm install
```

3. **Environment Setup**
```bash
# Backend environment (.env)
cd ecosphere-backend
cp .env.example .env

# Configure the following variables:
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_google_gemini_api_key
CLOUDINARY_CLOUD_NAME=your_cloudinary_cloud_name
CLOUDINARY_API_KEY=your_cloudinary_api_key  
CLOUDINARY_API_SECRET=your_cloudinary_api_secret
```

4. **Start the application**
```bash
# Start backend server (from ecosphere-backend directory)
npm run dev

# Start frontend (from root directory)
npm start
```

### 🌐 Application URLs
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000
- **API Documentation**: http://localhost:5000/api/docs
